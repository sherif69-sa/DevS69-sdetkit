# Partner-ready implementation pack (sample)

Use this pack when onboarding a new partner/customer organization to SDETKit.

## Pack contents

1. **Discovery worksheet**
   - target repositories
   - lane recommendation (Startup/Scale/Regulated)
   - compliance constraints
2. **Pilot plan template**
   - 30/60/90 milestones
   - named owner matrix
   - success KPI targets
3. **Enablement checklist**
   - canonical path run in pilot repos
   - dashboard and executive template activated
   - support/escalation model accepted
4. **Handoff checklist**
   - operations handbook ownership transfer
   - compatibility policy acknowledgment
   - rollback and incident communication protocol

## Suggested timeline

- Week 1: discovery + lane selection
- Week 2-3: pilot deployment and evidence capture
- Week 4: leadership readout and rollout decision

## Exit criteria

- Pilot KPIs baseline captured and reviewed.
- At least one weekly executive report produced.
- Support/escalation ownership and contacts confirmed.
